Erreur de compression

Une erreur est survenue lors de la compression.

Que faire ?

  • Vérifiez la taille : Maximum 50MB par fichier
  • Format supporté : PDF, JPG, PNG ou WebP uniquement
  • Limite atteinte : 5 fichiers maximum par heure
  • Fichier corrompu : Vérifiez que le fichier s'ouvre correctement
